Morse Code
==========

Encoding:
  HELLO -> .... . .-.. .-.. ---
  SOS -> ... --- ...
  TONIC -> - --- -. .. -.-.
  HI WORLD -> .... .. / .-- --- .-. .-.. -..
  42 -> ....- ..---

Decoding:
  .... . .-.. .-.. --- -> HELLO
  ... --- ... -> SOS
  - --- -. .. -.-. -> TONIC
  .... ..   .-- --- .-. .-.. -.. -> HI WORLD
  ....- ..--- -> 42
:ok
